Type Effectiveness Notes and Tips
✧ Normal Effectiveness is the regular type style while Inverse Effectiveness inverts the regular type chart, turning immunities and resistances into weaknesses, and weaknesses into resistances.
✧ If you want to check the type chart, use the Pokémon Deluge's Normal Type Chart and Inverse Type Chart.
✧ Type effectiveness greatly influences how much damage moves deal
  • Damage normally - Normal Damage (x1).
  • Weak to - Effective (x2) and Super Effective (x4).
  • Resistant to - Endure the damage (x1/2) and (x1/4).
  • Immune - No Damage or Invulnerable.
✧ For more details about type effectiveness, check the Bulbapedia's Type Effectiveness.
How to use Type Effectiveness?
✧ Select one or two types from the list of Pokémon Types.
✧ Clickbutton to display the result of type effectiveness.
✧ The switch on/off checkbox (Inverse Type Effectiveness) will show/hide the Inverse Type Effectiveness Mode.
